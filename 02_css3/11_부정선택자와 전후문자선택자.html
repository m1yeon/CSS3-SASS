1. 전후문자 선택자

CSS3에서 추가된 선택자로 가상클래스(:)와 가상요소 선택자(::)가 있음

가상클래스는 사용자의 행동(action)이나 시스템의 상태(status)등 외부적인 요인에 의해서 웹브라우저에서
html문서와 특정 태그들을 발생할 수 있는 특정한 상태를 CSS표준으로 정의해 놓은 것을 말하며,
대표적인 가상클래스는 :link :visited :hover :active :checked :focus :enabled :disabled가 있음

'가상요소'는 태그 내용의 특정 부분을 CSS표준으로 미리 지정한 것을 말하며
대표적인 가상요소에는 ::first-letter ::first-line ::before ::after등이 있음
가상클래스는 특정 부분에만 서식을 적용하기 위한 요소로 "::"기호를 사용함


가상 요소 선택자(::)

  • ::first-letter : 문단의 첫줄의 첫글자만 선택
  • ::first-line : 문단의 첫줄을 선택
  • ::before : 사용자가 선택한 요소의 앞쪽을 선택
  • ::after : 사용자가 선택한 요소의 뒤쪽을 선택
